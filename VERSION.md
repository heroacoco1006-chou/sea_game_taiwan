# 《大航海福爾摩沙》版本沿革

本專案以「六角格海戰正式啟用」作為 V1 與 V2 的分水嶺。版本號採 `主版本.次版本.修正版`：玩法世代改變升主版本，新功能升次版本，錯誤與平衡修正升修正版。

## V2.0.1｜港町與航向修正版（2026-07-19）

- 依試玩標記圖重新定位日本港町的酒館、旅館、官府、交易所與道具店，並延伸對應可走區域。
- 依試玩標記圖重新定位南洋港町的酒館、交易所、道具店與港口。
- 修正大地圖船隻向左、向右移動時使用相反方向圖片的問題；上下與斜向判定維持不變。

## V2.0.0｜六角海戰版（2026-07-19）

V2 從提交 `676b35d` 正式啟用六角格回合制海戰；`v2.0.0` 標籤則指向完成首輪線上試玩修正後的穩定版本。

主要內容：

- 正式採用 11×7 六角格戰場、六方向船首、側舷砲線、島嶼遮蔽與撤退邊界。
- 玩家實際擁有的 1～5 艘船全數進場；一般海盜依玩家成長逐步提高地圖、船數與船型。
- 支援移動、轉向、砲擊、接舷、修整、等待、撤退、反悔移動與條件式自動戰鬥。
- 新增三軌聲望、6 件一次性聲望特殊事件與 3 件唯一稀有道具。
- 納入任務接取港提示、入港清除海上負面狀態及港町／造船廠畫面修正。
- 舊海戰仍保留為緊急回退，可用網址參數 `?battle=legacy` 測試。

## V1.0.0｜經典海戰最終版（2026-07-15）

標籤固定指向提交 `2474f97`，即六角海戰已完成內部驗收、但尚未切換成正式入口之前的最後版本。

主要內容：

- 22 個港口與東亞、東南亞世界地圖。
- 三位主人翁、各 10 章主線、25 位夥伴與專屬任務。
- 自由貿易、探索、圖鑑、造船、艦隊、裝備、補給與海上事件。
- 採購、討伐、探索三類一般委託。
- 正式流程仍使用原本的經典回合制海戰。

## 發版約定

每次公開網站版本更新時，至少同步：

1. `src/version.ts` 的玩家可見版號與版本名稱。
2. `package.json`／`package-lock.json` 的版本號。
3. 本文件的版本說明。
4. Git annotated tag（例如 `v2.0.1`）並推送到 GitHub。

純內容或修正採 `v2.0.x`；增加一組完整新系統採 `v2.x.0`；再次更換核心玩法世代才升 `v3.0.0`。
