# 大航海福爾摩沙（Age of Sail: Formosa）

給國小五、六年級學生的 2D 角色扮演航海貿易遊戲，透過遊玩理解**台灣在大航海時代（1600～1662）的角色**。

- 玩法風格參考 KOEI《大航海時代2》（僅模仿風格，不使用任何其版權素材）
- 三位主人翁：漢人海商、荷蘭 VOC 商務員、日本朱印船商人之女
- 地圖以台灣為中心，涵蓋東亞與東南亞（含香料群島）22 個港口
- 系統：自由貿易、船隻購買／建造／改造、艦隊、回合制海戰、歷史圖鑑收集

## 如何執行（開發版）

```bash
npm install
npm run dev
```

然後用瀏覽器開 `http://localhost:5173`。

### 操作方式

- 方向鍵或 WASD：航行
- 靠近港口按 Enter：入港
- 滑鼠點擊：操作港口設施與交易所
- 旅館可休息＋存檔；入港時也會自動快速存檔
