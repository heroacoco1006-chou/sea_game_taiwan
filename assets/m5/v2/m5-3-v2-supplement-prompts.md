# M5-3 V2 補充素材 Prompt 紀錄

> 本檔記錄主角行走圖與船隻裝備外觀的 imagegen／image2.0 產圖方向，供後續重繪、補幀或統一風格時比對。

---

## 主角行走圖 v2

- 目標：三位主角（林海生、彼得・范德堡、田中千代）港町行走 sprite sheet。
- 風格：V2 精緻 2D 手繪航海 RPG 風格，17 世紀東亞海商時代感，角色可愛但不幼稚，線條清楚、服飾有文化差異。
- 構圖：三列角色，每列多格動畫，包含正面待機、正面踏步、側面行走、背面待機。
- 背景：純綠 chroma key，方便後續用 Pillow 去背輸出透明 PNG。
- 輸出：source 保留於 `assets/m5/v2/source/m5-3-hero-walk-v2-source.png`；切圖後輸出到 `assets/m5/v2/characters/walk/`。

## 船隻裝備外觀

- 目標：船首像 x4、裝甲 x3、船帆 x3、大砲種類 x3，另含船身、索具、砲艙預覽格。
- 風格：V2 精緻 2D 手繪航海 RPG 風格，舊羊皮紙背景、深胡桃框線、金色與朱紅點綴。
- 構圖：4x4 圖板，每格為清楚可辨識的裝備外觀，避免 UI 小尺寸時混淆。
- 內容：
  - 船首像：媽祖像、海龍像、獅子像、鳳凰像。
  - 裝甲：木製補強、鐵製包板、銅皮船底。
  - 船帆：棉布帆、帆布硬帆、多桅軟帆。
  - 大砲：輕型佛朗機砲、散彈砲、重型紅夷砲。
- 輸出：source 保留於 `assets/m5/v2/source/m5-3-ship-equipment-v2-source.png`；切圖後輸出到 `assets/m5/v2/ships/equipment/`。
