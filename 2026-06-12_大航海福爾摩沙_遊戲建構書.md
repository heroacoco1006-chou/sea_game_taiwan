### 設計原則（給小學生玩的遊戲）

- **看得懂**：所有文字白話、生難字詞附簡短說明；數字系統簡單（不出現複雜匯率、利息）
- **不血腥**：海戰只有船隻損壞與「投降／撤退」，沒有人物死亡畫面；海盜玩法定位為「亦商亦盜的時代背景」，劇情會引導反思
- **玩中學**：歷史知識藏在劇情、NPC 對話、港口介紹、圖鑑收集中，不是考試式灌輸
- **可分段玩**：隨時存檔，每段主線任務 15～30 分鐘可完成一個段落

## 2. 歷史背景設定（1600～1662）

遊戲時間軸貫穿整個台灣大航海時代，主線劇情依真實歷史事件推進：

| 年代 | 歷史事件 | 在遊戲中的呈現 |
|------|---------|---------------|
| 1602 | 荷蘭東印度公司（VOC）成立 | 背景設定、荷蘭主人翁的雇主 |
| 1604／1622 | 荷蘭兩度佔領澎湖，被明軍要求退出 | 主線事件：引出「荷蘭為何來台灣」 |
| 1624 | 荷蘭退出澎湖、於大員（今台南安平）建熱蘭遮城 | 主線關鍵章節 |
| 1626 | 西班牙佔領雞籠（基隆）、建聖薩爾瓦多城 | 主線事件：南荷北西對峙 |
| 1628 | 濱田彌兵衛事件（日荷在大員的貿易衝突） | 日本主人翁主線核心事件 |
| 1633 | 料羅灣海戰（鄭芝龍艦隊擊敗荷蘭艦隊） | 漢人主人翁主線高潮之一 |
| 1635～ | 荷蘭在台推動鹿皮、蔗糖出口；漢人移民來台開墾 | 貿易系統：台灣特產的由來 |
| 1642 | 荷蘭北上驅逐西班牙，統一全台據點 | 主線事件 |
| 1652 | 郭懷一事件（漢人移民反抗荷蘭） | 劇情事件：理解殖民統治下的移民處境 |
| 1661～1662 | 鄭成功攻台、熱蘭遮城圍城、荷蘭投降 | 三線共同的最終章 |

> 劇情寫作原則：大事件遵照史實，主人翁是「虛構小人物視角」穿梭其中（類似歷史小說手法），避免竄改歷史結局。

## 3. 三位主人翁設計

玩家開局三選一，各自代表一種職業路線與一個歷史視角。三條主線在不同年代切入，最終都匯流到 1661～1662 的熱蘭遮城之戰。

### 3-1. 林海生（漢人海商，海盜／海商路線）

- **背景**：1605 年生於福建泉州，父親是跑南洋的船商，因海禁與風暴家道中落。18 歲投靠笨港的顏思齊集團，後追隨鄭芝龍。
- **起點**：1623 年，月港（漳州），一艘小戎克船。
- **主線軸**：海盜集團學徒 → 顏思齊病逝、鄭芝龍崛起 → 接受明朝招撫的抉擇 → 料羅灣海戰 → 鄭芝龍降清後的迷惘 → 加入鄭成功、參與攻台。
- **學到的歷史**：明末海禁、海商與海盜一線之隔、鄭氏集團如何控制台海貿易、鄭成功攻台始末。
- **玩法傾向**：戰鬥與掠奪較強，可發海盜任務，也可走純貿易路線。

### 3-2. 彼得・范德堡（荷蘭 VOC 商務員，商人路線）

- **背景**：1602 年生於阿姆斯特丹，VOC 見習商務員，被派往東方尋找對中國貿易的據點。
- **起點**：1622 年，巴達維亞（今雅加達），一艘小型平底商船。
- **主線軸**：澎湖受挫 → 轉進大員建城 → 建立鹿皮、蔗糖、生絲轉口貿易網 → 濱田彌兵衛事件中斡旋 → 驅逐西班牙 → 郭懷一事件的衝擊 → 熱蘭遮城圍城戰中見證時代落幕。
- **學到的歷史**：VOC 是什麼、台灣為何成為荷蘭的轉運站、荷蘭如何經營台灣（贌社、王田、傳教）、原住民與荷蘭的互動。
- **玩法傾向**：貿易與經營最強，戰鬥較弱，靠賺價差與完成商會任務推進。

### 3-3. 田中千代（日本朱印船商人之女，冒險／貿易路線）

- **背景**：1610 年生於平戶，父親持幕府朱印狀跑台灣航線。父親在濱田彌兵衛事件後鬱鬱而終，千代女扮男裝接手家業（提供女性角色選擇）。
- **起點**：1628 年，平戶，一艘朱印船。
- **主線軸**：重啟台灣航線 → 追查父親當年在大員的遭遇（帶出濱田彌兵衛事件全貌） → 鎖國令逐步收緊的壓力 → 透過琉球維持貿易 → 與漢人、荷蘭角色的友誼 → 1661 年以民間商人身分見證熱蘭遮城之戰。
- **學到的歷史**：朱印船貿易、日本鎖國、琉球的中介角色、台灣作為中日轉口站（中國生絲↔日本白銀）。
- **玩法傾向**：均衡型，冒險與探索任務多（尋找航線、探訪各港）。

## 5. 遊戲系統設計

### 5-1. 角色系統

- **等級與六大能力值（2026-06-17 確認並實作，詳見 `2026-06-17_等級與能力值架構.md`）**：
  - 船長有等級（Lv1～50）與六大能力值：**統率、砲術、武勇、航海、知識、交涉**（各 1～99）。
  - 經驗來源：完成主線章節、商館委託（採購／海戰／探索）、海戰勝利、招募夥伴；升級時提示能力提升。
  - 升級時依各主角「成長傾向」**自動加點**（不需玩家手動配點，符合國小友善）；三主角起始值與傾向不同，形成不同強項（林海生偏交涉／航海、彼得偏知識／交涉、千代偏航海／交涉），弱項也會按比例緩慢成長。
  - 能力值接入既有玩法：砲術↑海戰火力、航海↑航速與抗暴風、統率↑接舷與船員耐性、交涉↑買賣價、知識↑探索發現。
  - **夥伴固定能力值**：25 位夥伴依星級與專長有固定六圍（不升級）；指派到對應幹部職位時，其能力值加成到艦隊——招募並善用高星專長夥伴是強化戰力的關鍵（呼應 §5-7）。
- 職業聲望三軸：**商人聲望／海盜惡名／冒險名聲**，影響 NPC 反應與可接任務（規劃中）
- 一般船員：酒館雇用無名水手（補足船隻必要人數）
- **夥伴 NPC 系統**：25 位具名夥伴可邀請入隊、擔任幹部職位（仿大航海時代4 的航海士設定），詳見 §5-7
- **個人裝備**：主角與夥伴可裝備武器、防具、飾品，詳見 §5-8

### 5-2. 航海系統

- 世界地圖即時航行（可暫停），考慮風向與洋流（簡化為「順風快、逆風慢」＋黑潮加速帶）
- 補給管理：糧食、水按日消耗，耗盡則船員士氣下降→壞血病事件（教學點：當時航海的風險）
- 海上事件：暴風（颱風季 7～9 月機率高）、海盜襲擊、漂流物、發現新航線，以及持續狀態事件（鼠患、壞血病、船上叛亂、思鄉病等）
- **持續狀態與消耗品**（老闆 2026-06-13 指示）：海上狀態會持續數日並造成不同影響，例如鼠患增加糧食消耗、暴風雨持續損傷船體、壞血病與思鄉病增加疲勞、船上叛亂降低航速；玩家可用對應消耗道具立刻解除（萊姆治壞血病、船貓治鼠患、祈禱藥水停止暴風雨等）
- 簡化但保留「計畫航線」的樂趣：港口間航程 1～7 天（遊戲內時間），讓小朋友體會「距離與風險」

### 5-3. 貿易系統（核心玩法）

- 貿易品約 24 種，分 6 類：絲織品（生絲、絲綢）、瓷器陶器、香料（胡椒、丁香、肉豆蔻）、食品（米、糖、茶）、原料（鹿皮、硫磺、蘇木、錫、銅）、貴金屬（白銀）等
- **每港只販售 5～7 種商品**（含當地特產，老闆指示）：想要的貨必須去產地買，玩家自然理解「貿易因稀缺而生」與各地特產（台灣＝蔗糖鹿皮硫磺、日本＝白銀銅漆器、香料群島＝丁香肉豆蔻……）；賣出則不限品項（任何港都收購）
- 每港有特產（產地便宜）與需求品（賣得貴），重現史實貿易線：
  - 中國生絲 → 日本換白銀（利潤最高的歷史航線）
  - 台灣鹿皮 → 日本（武士甲冑用）
  - 台灣蔗糖 → 日本、波斯方向（抽象化為高價收購）
  - 香料群島丁香肉豆蔻 → 全圖高價
- 行情波動＋酒館情報（「聽說長崎生絲缺貨」），教供需概念
- 大宗交易會微幅影響當地價格（買多漲價），防無腦刷錢

### 5-4. 船隻系統

- 船型 8 種（依勢力風格）：小戎克船、大戎克船、福船、朱印船、卡拉維爾帆船、克拉克帆船、平底商船（笛型船）、蓋倫帆船
- 各船參數：**總空間（固定）**、船速、耐久、砲位數、**最低水手數／水手上限**
- **船艙配比系統（老闆指示）**：每艘船總空間固定，玩家在造船廠調整「商品艙↔糧水艙」比例；糧水艙大→可航行天數長、商品艙大→賺錢效率高，間接決定可帶的水手人數，玩家必須找到平衡
- **最低水手人數**：低於最低人數→航速減半；低於最低人數的一半→航速降到最慢，營造缺人的緊張感
- **水手與疲勞**（M2.5 起實裝）：水手影響船速與糧食消耗；疲勞值隨航行天數增加（水手越少升越快），突發事件（壞血病等）也會增加；疲勞滿 100 水手減員，水手歸零＝Game Over（可由存檔點繼續）
- **船價平衡**（老闆 2026-06-13 指示）：小型船約數千到萬元，中型船進入數萬～數十萬，高級大型西洋船至少百萬元級，避免玩家跑兩輪貿易就買到最高級船
- **建造限制與工期**：造船廠不直接販售現成船；玩家先下訂建造並付款，到期後回同一港口取船。小型船約 3 天，大型船最多 30 天；各地造船廠只提供符合當地技術與勢力的船型，中國港口造中國船，日本港口造日本船，大員因荷蘭據點可造西洋船，東南亞殖民港依勢力提供不同西洋船型
- **旅館逗留**：旅館可用每天 100 兩的費用讓時間往前走，方便等待造船完工，同時讓疲勞歸零並存檔
- **改造**：加裝船砲、更換帆（速度↑）、貨艙改裝（容量↔砲位取捨）、船身加固（耐久↑）——全部用金錢升級
- **船隻裝備**：大砲（分砲種）與船首像等可購買裝備，提升船隻能力，詳見 §5-8
- 艦隊：最多 5 艘編隊，旗艦＋僚艦（僚艦可純載貨）

### 5-5. 海戰系統

- 仿大航海2 簡化版：海戰地圖回合制，指令＝砲擊／接舷（白刃戰以士氣值對拼，無血腥畫面）／機動／撤退
- 勝利條件：敵旗艦耐久歸零（敵投降）或敵撤退；戰敗＝損失部分貨物金錢，回最近港口（不會 Game Over）
- 對小學生的調整：戰鬥節奏快（5 分鐘內結束）、可選「自動戰鬥」、主線必要戰鬥附難度提示

## 7. 技術架構

> 白話說明：下面是「用什麼工具做」，全部是免費、安全、廣泛使用的開源工具。

| 項目 | 選擇 | 白話理由 |
|------|------|---------|
| 遊戲引擎 | **Phaser 3**（開源網頁遊戲引擎） | 2D 遊戲最成熟的免費引擎，開發快、資料多 |
| 桌面打包 | **Electron** | 把遊戲包成 Windows 安裝檔（.exe），雙擊即玩、離線可用。Discord、LINE 電腦版都是這技術 |
| 程式語言 | TypeScript | 錯誤少、好維護 |
| 存檔 | 本機 JSON 檔 | 存在玩家電腦裡，不需網路、不收集任何資料 |
| 資料驅動 | 港口／商品／船／任務全部寫成資料檔（JSON） | 之後調整內容（加港口、改價格）不用動程式 |

開發時先以瀏覽器測試（迭代快），里程碑完成後打包成 Windows 安裝檔交付。
